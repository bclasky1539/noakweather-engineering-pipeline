# Changelog

## Version 0.2.0 - Date: March 12, 2025

- Implemented architectural reorganization according to domain-driven design principles throughout the weather data
<br>processing pipeline.
- Developed and established the core domain layer with defined entities, value objects, and domain services.


## Version 0.1.5 - Date: March 9, 2025

- Refactored the weather processor scripts into three separate scripts, one for current weather,
<br>one for forecast and one for shared functionality between both
- Added a print processor script for both weather types to provide a quick way to print
<br>weather information to standard out
- Refactored the dataclass scripts for better functionality

## Version 0.1.4 - Date: March 6, 2025

- Refactored the dataclass scripts
- Refactored the weather processor script into two separate scripts for current weather and forecast
- Updated the README.md file

## Version 0.1.3 - Date: March 5, 2025

- Initial release providing the current weather and forecast that connects with openweathermap API
